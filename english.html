Privacy Policy

Scope

The present privacy policy has been developed to clarify how the application Expense Manager handles the user personal data. Personal data are those relative to information which identify a person as an individual.

Personal data

All financial data entered in the Expense Manager application by the user are saved in the user device memory and the developer cannot have any access to them.

No other personal user data are required by the Expense Manager application.

User data backup

In case the user make use of the functionality to backup the Expense Manager data, this backup can be done or on the same user device or on Google Drive in the memory space reserved to the user through his Google account. In both cases the Expense Manager developer has no access to the user saved data. For Google Drive terms of service and privacy policies refer to Google Drive Terms of Service.

Also in the case the user make use of the functionality to create a pdf file containing all transactions done in a certain period of time, the Expense Manager developer can have no access to that file.

Data used by third parties

Expense Manager app uses third parties services such as Google Firebase Analytics and Google AdMob Advertising.

Firebase Analytics collects data in anonymous form to evaluate how users use the Expense Manager application, how long they use it, how many pages they watch, from which location they act etc. These information are very useful to the developer to improve the app usability. The developer does not memorize these information and Firebase Analytics complies with Google Analytics for Firebase Use Policy.

The application Expense Manager makes use of advertisement banners through the Google AdMob Advertising service which can collect in anonymous form data relative to the user interests also through cookies or similar technologies. Expense Manager uses advertisement banners to fund the app development costs and to maintain its efficiency along time. AdMob compyies with Google Privacy and Terms.

Acceptance of the policy terms

By using the Expense Manager application the user agrees to the terms of the present policy. The continued use of the app after any policy update will be deemed acceptance of the new policy.

Policy update

The present policy can be changed in any time. It's user responsibility to review periodically this policy to acknowledge the new terms
